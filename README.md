# Nictionary

Nictionary is a self-contained offline-friendly five-letter word game. Guess the word in six tries, with Daily and Practice modes, separate statistics, backup/restore, and shareable results.

## Play online

Open `index.html` through GitHub Pages or any static web host.

## Play offline

Download `Nictionary_Offline.html` and open it directly in a modern browser. The game does not require a server or internet connection after download.

## Saves

Progress is stored in your browser local storage. Online and offline copies may have separate saves, so use **Settings → Export save data** and **Import save data** to move progress between copies.
